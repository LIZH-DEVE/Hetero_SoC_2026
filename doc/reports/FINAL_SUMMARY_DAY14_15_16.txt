系统数据流
================================================================================

完整数据流:
  RX → RX Parser → 5-Tuple Extractor → ACL Match Engine → PBM → Crypto Engine → TX Stack → Loopback → TX

安全路径:
  CPU → Config Auth → CSR → Key Vault → Crypto Engine → Data Path

各模块功能:
  1. RX Parser: 解析协议,检查长度和对齐
  2. 5-Tuple Extractor: 提取五元组
  3. ACL Match Engine: ACL匹配和丢弃
  4. PBM Controller: Ring Buffer,原子预留,回滚
  5. Crypto Engine: AES/SM4加密,CBC模式
  6. TX Stack: Checksum Offload,Padding
  7. DMA: 4K边界拆包,AXI Master
  8. Config Auth: 配置包认证,防重放
  9. Key Vault: DNA绑定,密钥派生,防克隆
